# Todo App
Est une application de gestion de tâches basiques entièrement faite en html css et js
Elle permet, après authentification d'ajouter, de mettre à jour et de supprimer des tâches et catégories
Chaque tâche est ajoutée à une catégorie
On peut cocher une tâche pour indiquer qu'elle est terminée
 On peut lister les tâches de chaque catégorie et les tâches terminées
